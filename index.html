Data Processing Projects

  • Judge Attitudes from Movie Reviews

  • Tweets Sentiment Analysis Project

    • Extracted the tweets, tagged each tweet into positive, negative, or neutral, and cleaned the text. Applied CountVectorizer and TfidfTransformer to transform the text contents into things could be quantified. Utilized Naive Bayes Classifier and Support Vector Machine Classifier to get the metric of accuracy, and used ten fold cross validation to improve the result. The main goal of this project is to learn how to extract tweets from twitter and processed the data.
